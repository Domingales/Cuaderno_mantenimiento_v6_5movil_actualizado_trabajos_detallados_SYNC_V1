========================================================
CUADERNO DE MANTENIMIENTO
Versión con sincronización entre dispositivos
========================================================

DESCRIPCIÓN GENERAL
--------------------------------------------------------

Cuaderno de Mantenimiento es una aplicación diseñada para
registrar, organizar y consultar trabajos de mantenimiento
realizados en instalaciones, edificios, fincas, industrias
o cualquier otro entorno donde sea necesario llevar un
control detallado de las tareas efectuadas.

La aplicación permite registrar cada intervención realizada,
las horas empleadas, materiales utilizados, ubicaciones,
observaciones y cualquier información relevante para
mantener un historial completo de los trabajos realizados.

Además, incorpora sincronización entre dispositivos mediante
Firebase, permitiendo compartir y mantener actualizados los
datos entre varios teléfonos o tablets.

--------------------------------------------------------
FUNCIONES PRINCIPALES
--------------------------------------------------------

1. REGISTRO DE TRABAJOS

Permite crear registros detallados de mantenimiento
incluyendo:

- Fecha.
- Horas trabajadas.
- Horas extra.
- Localidad.
- Ubicación.
- Tiempo empleado.
- Materiales utilizados.
- Observaciones.
- Trabajos realizados.

Cada registro queda almacenado para futuras consultas.

--------------------------------------------------------
2. TRABAJOS COMPLETADOS
--------------------------------------------------------

Dentro de cada registro es posible añadir varios trabajos
realizados.

Ejemplos:

- Cambio de filtro.
- Sustitución de lámpara.
- Reparación de fuga.
- Limpieza de cuadro eléctrico.
- Revisión de motor.
- Ajuste de maquinaria.

Los botones + y - permiten añadir o eliminar líneas de
trabajo según sea necesario.

--------------------------------------------------------
3. CONTROL DE HORAS
--------------------------------------------------------

La aplicación permite registrar:

- Horas ordinarias.
- Horas extra.

Esto facilita el seguimiento de la actividad laboral y la
posterior elaboración de informes.

--------------------------------------------------------
4. HISTORIAL DE REGISTROS
--------------------------------------------------------

Todos los trabajos guardados aparecen en la sección
REGISTROS.

Desde esta sección se pueden:

- Consultar registros anteriores.
- Revisar trabajos realizados.
- Editar registros existentes.
- Eliminar registros si es necesario.

--------------------------------------------------------
5. COPIAS DE SEGURIDAD
--------------------------------------------------------

La aplicación dispone de funciones de copia de seguridad
para evitar pérdidas de información.

Permite:

- Exportar datos.
- Importar datos.
- Restaurar información previamente guardada.

Se recomienda realizar copias periódicamente.

--------------------------------------------------------
6. SINCRONIZACIÓN ENTRE DISPOSITIVOS
--------------------------------------------------------

La aplicación incorpora sincronización mediante Firebase.

Características:

- Sincronización automática.
- Actualización de cambios entre dispositivos.
- Conservación de registros aunque se cambie de móvil.
- Recuperación rápida de información.

Para utilizar esta función es necesario configurar la
sincronización desde el apartado AJUSTES.

--------------------------------------------------------
7. AJUSTES
--------------------------------------------------------

Desde Ajustes es posible:

- Configurar la sincronización.
- Gestionar copias de seguridad.
- Ajustar parámetros de funcionamiento.
- Consultar información de la aplicación.

--------------------------------------------------------
MODO DE USO RECOMENDADO
--------------------------------------------------------

1. Crear un nuevo registro.
2. Introducir la información del trabajo realizado.
3. Añadir los trabajos completados.
4. Registrar materiales utilizados.
5. Guardar el registro.
6. Comprobar que aparece en el historial.
7. Sincronizar periódicamente para mantener los datos
   actualizados entre dispositivos.
8. Realizar copias de seguridad regularmente.

--------------------------------------------------------
OBJETIVO DE LA APLICACIÓN
--------------------------------------------------------

El objetivo principal de Cuaderno de Mantenimiento es
disponer de una herramienta sencilla y eficaz para llevar
un control detallado de todos los trabajos de mantenimiento
realizados, permitiendo consultar el historial de
intervenciones, controlar horas trabajadas y conservar toda
la información organizada y accesible desde distintos
dispositivos.
